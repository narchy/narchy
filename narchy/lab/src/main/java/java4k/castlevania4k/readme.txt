Castlevania 4k
==============

Description
-----------

Assuming the role of Simon Belmont, you enter Count Dracula's chambers with the Morning Star Vampire Killer Whip, 
the Cross Boomerang, the Triple Shot, unlimited hearts and unlimited lives.

Instructions
------------

Left and Right -- Move
Down -- Kneel
S -- Whip
A -- Boomerang
Up or D or any other key -- Jump
