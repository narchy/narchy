23 seconds
==========

Description
-----------

Lost in space with defective laser weapons, your only way to survive are the ship's bombs.
But these need 23 seconds to reload. 23 seconds you need to survive.

Instructions
------------

Arrows: Move
X/Space: Bomb
Z/Y: Speed up
Left Shift: Slow down
Escape: Abort game

Scoring:
Destroy as many enemies as possible at once.
Graze bullets.
Collect the small items enemies drop when being destroyed.
Collect the large items that appear from time to time. Don't miss them to get more points.
(By the way: Destroying enemies immediately after you've died, doesn't give you many points)
