My Precious
===========

Description
-----------

MyPrecious is a turn based puzzle platformer.

It is a 4k version of the great "Zuki's Quest" game by Tinytouchtales.

Instructions
------------

Play the game with cursor keys and collect all golden skulls.

Press 'r' to restart the level
Press 'n' to start the next level
Press 'p' to start the previous level
